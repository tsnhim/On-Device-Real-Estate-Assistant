# Main Task

Build and document an on-device real-estate assistant that:

- accepts spoken or typed user input
- uses Whisper to transcribe speech when needed
- uses retrieval assets to fetch relevant supporting context
- uses FLAN-T5 to generate the final answer

The current repo keeps the model artifacts and retrieval data needed for that direction, while future work should focus on reproducible inference, evaluation, and device benchmarking.
